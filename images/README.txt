========================================
    收款二维码图片说明
========================================

请在此目录放置以下两个二维码图片：

1. wechat-qrcode.jpg
   - 微信收款二维码
   - 建议尺寸：400x400像素
   - 格式：JPG或PNG

2. alipay-qrcode.jpg
   - 支付宝收款二维码
   - 建议尺寸：400x400像素
   - 格式：JPG或PNG

========================================
    如何获取收款二维码
========================================

【微信收款码】
1. 打开微信 → 我 → 服务 → 收付款
2. 点击"二维码收款"
3. 保存收款码图片
4. 重命名为 wechat-qrcode.jpg
5. 放入此目录

【支付宝收款码】
1. 打开支付宝 → 首页 → 收钱
2. 点击"保存图片"
3. 重命名为 alipay-qrcode.jpg
4. 放入此目录

========================================
    注意事项
========================================

- 如果没有放置二维码图片，页面会显示客服联系方式
- 建议定期更换收款码，确保安全
- 二维码图片不要太大，建议不超过500KB
